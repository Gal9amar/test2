# 🚨 בוט אזעקות פיקוד העורף — טלגרם

בוט טלגרם לקבלת התראות אזעקה בזמן אמת + מפה אינטראקטיבית.

## פקודות
| פקודה | תיאור |
|-------|--------|
| `/start` | הרשמה להתראות |
| `/stop` | הפסקת התראות |
| `/now` | אזעקות פעילות כרגע |
| `/stats` | סטטיסטיקות 24 שעות |
| `/city [עיר]` | חיפוש לפי עיר |

## פריסה על Railway
הוסף Environment Variable:
```
BOT_TOKEN = <הטוקן שלך>
```
